# PyTorch Beginner Path 🚀

A structured path for beginners to learn PyTorch step-by-step by coding along mainly to understand the concept.

## 📌 Goals

To help you understand the fundamentals of PyTorch through simple, progressive examples.

## 📂 Structure (so far)

* `pytorch1.ipynb` → PyTorch basics (tensors, operations with simple model)

More notebooks will be added in order.

## 🧠 How to Use

1. Go through the notebooks in order (`pytorch1`, `pytorch2`, ...)
2. Run the code and try to understand each step
3. Modify and experiment to reinforce learning

## 🔄 Updates

New content will be added to expand this path into more advanced topics.

---

I hope you follow along and build your PyTorch skills step by step.
